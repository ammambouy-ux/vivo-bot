# Telegram Bot 🤖

## Как получить токен у BotFather

1. Открой Telegram и найди [@BotFather](https://t.me/botfather)
2. Напиши ему `/newbot`
3. Придумай имя боту (например, `Мой классный бот`)
4. Придумай username, заканчивающийся на `bot` (например, `moy_klassny_bot`)
5. BotFather пришлёт тебе **токен** — длинную строку вида:
   ```
   1234567890:ABCdefGHIjklmNOPqrstUVWxyz-abcdefghij
   ```
6. Скопируй этот токен

## Как запустить бота

### Способ 1: Через переменную окружения (безопасно)

```bash
export BOT_TOKEN="твой_токен_сюда"
python3 bot.py
```

### Способ 2: Вставить токен прямо в код

Открой `bot.py`, найди строчку:
```python
BOT_TOKEN = os.environ.get("BOT_TOKEN", "ТВОЙ_ТОКЕН_СЮДА")
```
Замени `"ТВОЙ_ТОКЕН_СЮДА"` на свой токен и запусти:

```bash
python3 bot.py
```

## Команды бота

- `/start` — приветствие
- `/help` — список команд
- `/info` — информация о тебе
- Любой текст — бот повторяет его (эхо)

## Как остановить бота

Нажми `Ctrl + C` в терминале.
